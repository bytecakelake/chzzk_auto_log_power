치지직(chzzk.naver.com) 라이브 방송에서 통나무(파워) 자동 획득 및 보유 파워 개수, 채널별 파워 랭킹을 표시해주는 크롬 확장 프로그램입니다.

## 주요 기능
- 통나무(파워) 자동 클릭 및 획득
- 보유 파워 개수 표시 (채팅 입력창 옆)
- 채널별 통나무 파워 랭킹 팝업 (아이콘 클릭 시)
